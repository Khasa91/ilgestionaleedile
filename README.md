# AccessoCantieri.com

Boilerplate repository for the multi-tenant SaaS dedicated to construction companies. It includes a basic backend and frontend setup plus design documentation.

## Structure

- `docs/` – design notes and roadmap.
- `backend/` – Express + TypeScript API skeleton.
- `frontend/` – Next.js + Tailwind UI skeleton.

See `docs/DESIGN.md` for schema, wireframes and user flows.
